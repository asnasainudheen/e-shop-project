# e-shop-project
Full-stack E-Shop application built using HTML, CSS, JavaScript, Node.js, Express, and MongoDB. Features include user registration/login, product browsing, shopping cart management, order placement, and order tracking.

## Features

- User Registration & Login
- Product Catalog
- Shopping Cart
- Order Placement
- Order Tracking
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
